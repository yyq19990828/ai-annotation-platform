## [0.9.3] - 2026-05-07 · 前端杂项收口（Refactored Lighthouse）

主项详情见 [`CHANGELOG.md` → 0.9.3](../../CHANGELOG.md)。要点速览：

- **API 密钥**：完整后端（`api_keys` 表 + `/me/api-keys` CRUD + `ak_` token 走 `get_current_user`）+ 前端 `ApiKeysModal`（list / create 一次性显明文 / revoke）。
- **超管 ML 集成总览**：新页 `/admin/ml-integrations` 只读聚合 storage health + 跨项目 ml_backends（仅 super_admin）。
- **登录 progressive CAPTCHA**：同 IP 失败 ≥ 5 次后强制 Turnstile，401 加 `X-Login-Failed-Count` header；正常用户零打扰；dev 短路。
- **IoU rbush 索引**：同类分桶 RBush + 候选裁剪 + iouShape 精确判定 + some() 早退保留；千框场景预热。
- **DropdownMenu 收编**：通用组件加 `content` 自定义槽，把 ExportSection / NotificationsPopover 收到统一浮层骨架。
- **测试**：后端新增 9 case（api_keys / admin_ml_integrations / progressive_captcha），前端新增 6 case（iou-index / DropdownMenu content）。

详细计划：[`docs/plans/2026-05-07-v0.9.3-phase1-refactored-lighthouse.md`](../../docs/plans/2026-05-07-v0.9.3-phase1-refactored-lighthouse.md)。

## [0.9.2] - 2026-05-07 · M2 工作台 `S` 工具 + 文本入口（Luminous Canvas）

主项详情见 [`CHANGELOG.md` → 0.9.2](../../CHANGELOG.md)。要点速览：

- 工作台新增 `S` 工具（点 / Alt+点 / 拖框）+ AI 助手 SAM 文本提示输入框；候选 polygon 紫虚线叠加 Konva，`Enter` / `Esc` / `Tab` 接受 / 取消 / 切换。
- `useInteractiveAI` 集中路由 + 80ms 防抖 + inflight 计数防过期请求覆盖。
- ProjectSettings 加 DINO `box_threshold` / `text_threshold` 项目级 override（默认 0.35 / 0.25），`/predict` 协议透传。
- 一致的紫色虚线视觉语言（拖框预览 + 候选 polygon + 视觉同源），与 BboxTool 工具流完全隔离（独立 `samProbe` DragInit）。
- 测试：vitest 349 / pytest 277 全过；新增 `useInteractiveAI` 10 case + interactive 阈值注入 3 case。E2E 推迟到下版本（需要 `/_test_seed` ML backend 工厂）。

详细计划：[`docs/plans/2026-05-07-v0.9.2-luminous-canvas.md`](../../docs/plans/2026-05-07-v0.9.2-luminous-canvas.md)。
