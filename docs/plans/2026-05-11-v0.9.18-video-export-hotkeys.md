# v0.9.18 视频导出闭环 + 视频快捷键中心化

> 日期：2026-05-11
> 分支：`0.9.18`
> 范围来源：`ROADMAP/2026-05-11-video-workbench.md` M4 + `ROADMAP/2026-05-11-video-workbench-vs-image-gap.md` O1

## 1. 目标

完成 `video-track` 的首版导出闭环，并把 `VideoStage` 自有快捷键迁移到工作台中心化快捷键体系。

成功标准：

1. `video-track` 项目通过现有 `format=coco` 入口导出 Video Tracks JSON，默认只导出关键帧，可选展开所有帧。
2. 批次导出同样支持视频 JSON，并保留 task/batch 过滤语义。
3. `format=yolo|voc` 对视频项目返回清晰 400，避免误导为图片检测格式。
4. Dashboard 导出入口对视频项目只展示 Video JSON，并传递 `keyframes|all_frames`。
5. 视频快捷键由 `hotkeys.ts` / `useWorkbenchHotkeys` 统一分发，`VideoStage` 不再注册全局 `keydown`。
6. 帮助面板能展示视频快捷键。
7. 文档、CHANGELOG、Roadmap 同步更新。

## 2. 不做范围

- 不做撤销重做、offline/conflict diff、属性面板、多选或复制粘贴。
- 不做视频 AI tracker / SAM video。
- 不新增数据库表或迁移。
- 不把视频导出伪装成 COCO；只复用 `format=coco` 兼容入口，响应用 `export_type: "video_tracks"` 明确声明格式。

## 3. 后端实现

### 3.1 API 参数

- `GET /api/v1/projects/{id}/export?format=coco`
- `GET /api/v1/projects/{id}/batches/{batch_id}/export?format=coco`
- 新增可选查询参数：`video_frame_mode=keyframes|all_frames`
- 默认值：`keyframes`
- 图片项目忽略该参数。

### 3.2 Video Tracks JSON

导出结构包含：

- `export_type`
- `exported_at`
- `frame_mode`
- `project`
- `categories`
- `tasks`
- `tracks`
- `keyframes`
- legacy `video_bbox`
- `video_metadata`

`include_attributes=false` 时：

- 不输出 attribute schema。
- 不输出 annotation / track / keyframe 上的 attributes。

### 3.3 all_frames 展开

- 后端按相邻关键帧线性插值 `x/y/w/h`。
- `absent=true` 关键帧阻断跨段插值。
- 缺少 `frame_count` 时，以最大已标注帧兜底。

## 4. 前端实现

- `projectsApi.exportProject` / `batchesApi.exportBatch` 增加 `videoFrameMode` 选项。
- Dashboard 表格与卡片导出对 `video-track` 只展示 Video JSON。
- 视频导出浮层提供 `keyframes` / `all_frames` 选择。
- 图片项目保持 COCO / VOC / YOLO 和“包含属性数据”原行为。

## 5. 快捷键实现

- `VideoStage` 改为 `forwardRef`，暴露：
  - `togglePlayback()`
  - `seekByFrames(delta)`
- `useWorkbenchHotkeys` 接收 `videoMode` 和 `videoControlsRef`。
- 视频模式分发：
  - `Space` 播放 / 暂停
  - `ArrowLeft` / `ArrowRight` 逐帧
  - `Shift+ArrowLeft` / `Shift+ArrowRight` 跳 10 帧
  - `Delete` / `Backspace` 删除选中轨迹
  - `Tab` / `Shift+Tab` 循环 track
  - `Esc` 取消选择
  - `1-9` 切 active class
- 视频模式下禁用图片 nudge / SAM / polygon 等冲突快捷键。
