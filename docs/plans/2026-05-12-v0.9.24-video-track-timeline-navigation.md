# v0.9.24 · 视频时间轴可视化与关键帧导航

> 日期：2026-05-12
> 范围来源：`ROADMAP/2026-05-12-video-workbench-rendering-optimization.md` R4.1-R4.3

## 1. 背景

v0.9.23 已经把 `video_track.outside` 段和 timeline marker 数据打通，但 `VideoPlaybackOverlay` 仍是 seekbar 上叠少量 marker。用户在密集轨迹场景里仍然难以判断当前 track 的 keyframe、outside、prediction 和插值段分布。

本版把 R4 拆成较小的第一版：先完成选中轨迹时间轴、未选中全局密度条和关键帧跳转；R4.4 loop region、R17 hover 缩略图、R18 多速率播放、R19 bookmark 继续留到后续版本。

## 2. 目标

1. 选中 `video_track` 时，时间轴展示该 track 的 keyframe 点、outside 灰段、interpolated 虚线段和 prediction 标记。
2. 未选中 track 时，时间轴展示全部 track 的密度热度条，帮助快速找到有标注的帧段。
3. `Shift+←/→` 在选中 track 时跳到上/下一个可见 keyframe；未选中 track 时保持现有 ±10 帧跳转。
4. 插值段不跨 `outside` 和旧 `absent=true`；prediction keyframe 使用独立视觉状态。
5. 保留现有 seekbar、播放、逐帧控制和 hover 帧号 tooltip，不重写播放状态。

## 3. 不做范围

- 不做 R4.4 loop region。
- 不做 hover thumbnail、章节、已下载范围可视化。
- 不做时间轴 range 拖拽编辑 outside。
- 不改后端 schema/API，不新增数据库迁移。
- 不改变 `video_track.outside` 协议。

## 4. 实现方案

### 4.1 时间轴数据 helper

新增 `apps/web/src/pages/Workbench/stage/videoTrackTimeline.ts`：

- `buildSelectedTrackTimeline(track)`：输出单轨 timeline 模型，包括 keyframe marker、outside segment、interpolated segment。
- `buildGlobalTimelineDensity(tracks, maxFrame, bins = 80)`：按固定 bin 聚合 keyframe 数，供未选中 track 时显示密度条。
- `visibleKeyframesForTimeline(track)`：过滤 `absent` 和落在 outside 内的 keyframe，供关键帧跳转复用。

规则：

- outside 使用 v0.9.23 的 `effectiveOutsideRanges()`，兼容旧 `absent=true`。
- interpolated segment 只连接相邻可见 keyframe，且中间没有 outside 覆盖。
- prediction marker 由 keyframe `source === "prediction"` 决定。

### 4.2 Overlay UI

改造 `VideoPlaybackOverlay`：

- 新增 `selectedTrackTimeline` 和 `globalTimelineDensity` props。
- 选中 track 时渲染单轨层；未选中时渲染全局密度条。
- 保留 `timelineMarkers` 作为兼容输入，避免一次性删除 v0.9.23 的轻量 marker 路径。
- 使用稳定尺寸和绝对定位，避免 marker 增减导致 overlay 布局跳动。

### 4.3 关键帧跳转

改造 `VideoStage` 与 hotkey 接线：

- `VideoStageControls` 增加 `seekToKeyframe(dir: -1 | 1)`。
- `VideoStage` 根据当前选中 `video_track` 的可见 keyframe 计算目标帧，跳过 outside/absent 帧。
- `dispatchKey()` 在 `videoMode && hasSelectedVideoTrack && Shift+ArrowLeft/Right` 时返回 `videoSeekKeyframe`；否则继续返回原有 `videoSeek ±10`。
- 快捷键面板文案说明双语义：选中轨迹跳关键帧，否则跳 10 帧。
