# 标注工作台 — 界面与快捷键

工作台是日常标注的主要工作场所，由 4 个区域组成：

```
┌──────────────────────────────────────────────────────────────┐
│  TopBar：项目名 · 任务进度 · 提交按钮 · 用户菜单              │
├────────┬────────────────────────────────────┬────────────────┤
│        │                                    │                │
│ 工具栏 │           画布（Konva）            │   属性面板     │
│        │                                    │                │
│ - 平移 │   显示当前任务图像                 │ - 类别选择     │
│ - Bbox │   绘制 / 编辑标注                  │ - 标签         │
│ - 多边 │                                    │ - 备注         │
│        │                                    │                │
├────────┴────────────────────────────────────┴────────────────┤
│  底部：标注列表 · 撤销/重做 · 保存草稿 · 提交                 │
└──────────────────────────────────────────────────────────────┘
```

工具：`B` 矩形 · `P` 多边形 · `V` 平移。完整快捷键列表见下方（按工作台代码 SoT 自动生成）。

## 自动保存

每 30 秒自动保存草稿；意外刷新可恢复。

## AI 预标注

如果项目开启 AI 预标注，进入任务时已有候选框（紫色高亮）：

- 选中候选 → 修正位置/类别 → `A` 采纳
- 误识别 → `D` 驳回
- 漏标 → 切换工具自行标注

> 紫色 = AI 来源；蓝色 = 人工来源。
