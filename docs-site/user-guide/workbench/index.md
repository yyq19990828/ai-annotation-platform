# 标注工作台 — 界面与快捷键

工作台是日常标注的主要工作场所，由 4 个区域组成：

```
┌──────────────────────────────────────────────────────────────┐
│  TopBar：项目名 · 任务进度 · 提交按钮 · 用户菜单              │
├────────┬────────────────────────────────────┬────────────────┤
│        │                                    │                │
│ 工具栏 │           画布（Konva）            │   属性面板     │
│        │                                    │                │
│ - 移动 │   显示当前任务图像                 │ - 类别选择     │
│ - Bbox │   绘制 / 编辑标注                  │ - 标签         │
│ - 多边 │                                    │ - 备注         │
│ - 点   │                                    │                │
│        │                                    │                │
├────────┴────────────────────────────────────┴────────────────┤
│  底部：标注列表 · 撤销/重做 · 保存草稿 · 提交                 │
└──────────────────────────────────────────────────────────────┘
```

## 完整快捷键

> 真值来源：`apps/web/src/pages/Workbench/state/hotkeys.ts`，下表与代码同步。

### 工具切换

| 键 | 工具 |
|---|---|
| `W` | 选择/移动 |
| `R` | 矩形 (Bbox) |
| `P` | 多边形 (Polygon) |
| `K` | 关键点 |

### 画布

| 键 | 动作 |
|---|---|
| `Space` (按住) | 拖动画布 |
| `滚轮` | 缩放 |
| `0` | 适应窗口 |
| `1` | 100% 实际像素 |

### 编辑

| 键 | 动作 |
|---|---|
| `Ctrl/Cmd + Z` | 撤销 |
| `Ctrl/Cmd + Shift + Z` | 重做 |
| `Delete` / `Backspace` | 删除选中标注 |
| `Esc` | 取消当前操作 |

### 提交

| 键 | 动作 |
|---|---|
| `Ctrl/Cmd + S` | 保存草稿 |
| `Ctrl/Cmd + Enter` | 提交并加载下一条 |

## 自动保存

每 30 秒自动保存草稿；意外刷新可恢复。

## AI 预标注

如果项目开启 AI 预标注，进入任务时已有候选框（紫色高亮）：

- 选中候选 → 修正位置/类别 → 接受
- 误识别 → 选中 → `Delete`
- 漏标 → 切换工具自行标注

> 紫色 = AI 来源；蓝色 = 人工来源。
