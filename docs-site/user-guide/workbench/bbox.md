# Bbox 标注

## 操作

1. 按 `R` 切到矩形工具
2. 在画布上按下鼠标 → 拖动 → 松开，即生成一个矩形
3. 右侧属性面板选择类别

## 编辑已有 Bbox

- 单击 → 选中（出现 8 个角控制点）
- 拖角 → 缩放
- 拖中心 → 移动
- 双击边界 → 进入精确编辑模式

## 常见错误

- **画框太小**：低于 8×8 像素的框会被拦截，提示重画
- **超出图像**：会被自动 clip 到图像范围内
- **类别未选**：提交前必须为每个 Bbox 指定类别

## IoU 与质量

审核环节会用 IoU（Intersection over Union）判断你画的框与基准的重合度。一般要求 IoU ≥ 0.7 视为合格。
