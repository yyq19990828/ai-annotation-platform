# 视频追踪标注

视频任务在同一个工作台里打开，左侧队列、顶部提交、右侧属性与评论仍沿用图片工作台。画布区域会切换为视频播放器和时间轴。

## 基本操作

1. 打开 `video-track` 项目的任务。
2. 用底部时间轴或 `←` / `→` 定位到目标帧。
3. 暂停视频，在当前帧拖拽绘制 bbox。
4. 需要修正时，单击当前帧的框后拖动框体移动；按 `Delete` 删除。
5. 标完后点击「提交质检」。

## 播放与逐帧

| 操作 | 说明 |
|---|---|
| `Space` | 播放 / 暂停 |
| `←` / `→` | 后退 / 前进 1 帧 |
| `Shift + ←` / `Shift + →` | 后退 / 前进 10 帧 |
| 时间轴拖动 | 直接跳到指定帧 |

播放时画布只读，暂停后才能编辑，避免拖框和播放状态互相抢焦点。

## 当前版本边界

v0.9.16 保存的是逐帧 `video_bbox`，每个框只属于一个 `frame_index`。本版本还不支持轨迹 ID、关键帧插值、目标消失段、视频 AI tracker 或 SAM video predictor。

如果同一个对象出现在多帧，需要在这些帧分别画框。后续版本会把这些逐帧框升级为 track/keyframe 模型。
