# 用户手册

面向标注员、审核员、项目管理员的使用文档。

## 角色与权限

| 角色 | 主要职责 | 关键页面 |
|---|---|---|
| **超级管理员** (super_admin) | 全平台管理、用户/角色配置、系统监控 | 后台管理、审计日志 |
| **项目管理员** (project_admin) | 创建项目、上传数据、配置标注规范、分配任务 | 项目管理、数据集 |
| **标注员** (annotator) | 接受任务、完成标注 | 任务列表、标注工作台 |
| **审核员** (reviewer) | 审核已提交标注、回退/通过 | 审核工作台 |

## 推荐阅读顺序

1. [快速开始](./getting-started) — 第一次进入系统该看的
2. 按角色找对应章节：
   - 标注员 → [标注工作台](./workbench/)
   - 项目管理员 → [项目与批次](./projects/)
   - 审核员 → [审核流程](./review/)
   - 超级管理员 → [超管手册](./superadmin/)
3. 遇到问题查 [FAQ](./faq)

## 反馈与支持

应用右下角「BUG 反馈」按钮可一键提交问题，会自动附带当前页面、最近 API 调用、console 报错。
