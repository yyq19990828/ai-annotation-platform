# FAQ

## 登录与账号

**Q: 忘记密码？**
点击登录页「忘记密码」，会发邮件链接重置。如未配置邮件服务，联系超级管理员手动重置。

**Q: 账号被锁定？**
连续 5 次密码错误会被临时锁定 15 分钟；持续锁定请联系管理员。

## 标注

**Q: 提交后能否撤回？**
- 状态 `submitted` 但还没被审核 → 可在「我的任务」撤回
- 已审核完成 → 不可撤回，需要联系管理员

**Q: 浏览器卡顿？**
- 单图标注 > 200 个对象时卡顿正常，可关闭「显示标注列表」侧栏
- 视频标注帧率高时建议用 Chrome / Edge

**Q: 草稿丢失？**
草稿每 30 秒自动保存到本地 IndexedDB + 服务器。如果页面崩溃，重新进入工作台会提示恢复。

## 审核

**Q: 我能否回退已通过的任务？**
- 项目管理员可以
- 普通审核员不能

**Q: IoU 怎么算的？**
平台基于标准 Intersection over Union 计算，对多个对象按 Hungarian 匹配后取平均。

## 数据与权限

**Q: 我能看到别人的标注吗？**
- 标注员只能看到自己被分配的任务
- 审核员能看到本项目所有已提交标注
- 项目管理员能看到全部
- 超级管理员还能跨项目检索

**Q: 如何举报 BUG？**
应用右下角浮动按钮「BUG 反馈」会自动附带页面、API、console，无需截图。

## 性能

**Q: AI 预标注需要多久？**
- 一张图 ~ 2-5 秒（GroundingDINO + SAM 流水线）
- 大批量数据建议晚上批量预标，白天直接修正

## 还有问题？

提交 BUG 反馈或联系系统管理员。
